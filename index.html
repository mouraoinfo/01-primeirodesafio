HTML (HyperText Markup Language) é a base da construção de páginas web. As tags HTML são usadas para definir a estrutura e o conteúdo das páginas, e conhecê-las bem é essencial para qualquer desenvolvedor web. Neste guia, vamos explorar algumas das tags HTML mais comuns e mostrar exemplos práticos de como usá-las.

Cabeçalhos: <h1> até <h6>

Voltar ao início

Os cabeçalhos são usados para definir títulos e subtítulos. Eles variam de <h1> (mais importante) até <h6> (menos importante).

<h1>Este é um título de nível 1</h1>
<h2>Este é um título de nível 2</h2>
<h3>Este é um título de nível 3</h3>
<h4>Este é um título de nível 4</h4>
<h5>Este é um título de nível 5</h5>
<h6>Este é um título de nível 6</h6>

Parágrafos: <p>

Voltar ao início

A tag <p> é usada para definir parágrafos de texto.

<p>Este é um parágrafo de exemplo.</p>

Este é um parágrafo de exemplo.

Marcação de Texto: <mark>, <small>, <i>, <u>, <strong>, <del>

Voltar ao início

Estas tags são usadas para estilizar o texto de diferentes maneiras.

<p>Este texto tem uma <mark>marcação</mark> especial.</p>
<p>Este texto é <small>menor</small> que o normal.</p>
<p>Este texto está em <i>itálico</i>.</p>
<p>Este texto está <u>sublinhado</u>.</p>
<p>Este texto está em <strong>negrito</strong>.</p>
<p>Este texto foi <del>deletado</del>.</p>

Este texto está em itálico.

Este texto está sublinhado.

Listas Ordenadas e Não Ordenadas: <ol>, <ul>, <li>

Voltar ao início

Listas ajudam a organizar o conteúdo.

<ol>
  <li>Item 1 de uma lista ordenada</li>
  <li>Item 2 de uma lista ordenada</li>
</ol>

<ul>
  <li>Item 1 de uma lista não ordenada</li>
  <li>Item 2 de uma lista não ordenada</li>
</ul>
  1. Item 1 de uma lista ordenada
  2. Item 2 de uma lista ordenada
  • Item 1 de uma lista não ordenada
  • Item 2 de uma lista não ordenada

Links: <a>

Voltar ao início

A tag <a> é usada para criar hyperlinks.

<a href="https://example.com">Visite o Example.com</a>

Visite o Example.com

Linha Horizontal: <hr>

Voltar ao início

A tag <hr> é usada para inserir uma linha horizontal, geralmente para separar seções de conteúdo.

<hr>

Subscrito e Sobrescrito: <sub>, <sup>

Voltar ao início

Estas tags são usadas para texto subscrito e sobrescrito, respectivamente.

<p>H<sub>2</sub>O é a fórmula da água.</p>
<p>E=mc<sup>2</sup> é a famosa equação de Einstein.</p>

H2O é a fórmula da água.

E=mc2 é a famosa equação de Einstein.

Citação: <blockquote>

Voltar ao início

A tag <blockquote> é usada para citar blocos de texto de outras fontes.

<blockquote>Esta é uma citação famosa de uma pessoa importante.</blockquote>
Esta é uma citação famosa de uma pessoa importante.

Fonte: <font>

Voltar ao início

A tag <font> é usada para definir a cor, o tamanho e o tipo de fonte do texto. Nota: esta tag está obsoleta e é melhor usar CSS para estilizar fontes.

<font color="blue" size="4" face="Arial">Texto estilizado com a tag font</font> Texto estilizado com a tag font

Abreviações: <abbr>

Voltar ao início

A tag <abbr> é usada para definir abreviações ou acrônimos, fornecendo uma descrição mais detalhada quando o usuário passa o mouse sobre o texto.

<p><abbr title="HyperText Markup Language">HTML</abbr> é a linguagem de marcação usada para criar páginas web.</p>

HTML é a linguagem de marcação usada para criar páginas web.
